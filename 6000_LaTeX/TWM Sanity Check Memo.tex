I was worried about the TWM results, so I ran the a series of regressions in efforts to isolate the issue. \\

\begin{itemize}
	\item main: main regression
	\item TWM1: TWM without $post*(xbar_t - xbar) + post*(xbar_i - xbar)$ pieces

	\item main0: main regression, without controls
	\item TWM0: TWM without $post*(xbar_t - xbar) + post*(xbar_i - xbar)$ pieces, without controls

	\item main2: main regression, strongly balanced
	\item TWM2: TWM without $post*(xbar_t - xbar) + post*(xbar_i - xbar)$ pieces, strongly balanced

	\item TWM3: TWM without $post*(xbar_t - xbar) + post*(xbar_i - xbar)$ pieces, strongly balanced and without controls
	\item main3: main regression, strongly balanced and without controls
\end{itemize}
